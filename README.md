# Heart Disease Prediction App

## Project Overview
This is a Machine Learning web application built using Streamlit that predicts whether a person has heart disease based on medical parameters.

## Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Streamlit
- Pandas
- NumPy

## Features
- User input medical data
- ML model prediction
- Simple web interface
- Real-time prediction

## How to Run the Project

1. Clone the repository
2. Install requirements
3. Run Streamlit app

Command:
streamlit run app.py

## Project Structure
app.py – Streamlit frontend  
train_model.py – Model training  
model.pkl – Trained model  
requirements.txt – Libraries  

## Output
Heart Disease / No Heart Disease Prediction
